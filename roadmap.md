# alphaos roadmap

план развития операционной системы.

## v1.x — стабилизация и fat16
текущий этап. улучшение работы с текущей файловой системой.

- [ ] **поддержка папки (directories)**
    - [ ] реализация команды `cd` (смена директории).
    - [ ] чтение содержимого подпапок в `ls`.
    - [ ] поддержка путей вида `dir/file.txt`.
- [ ] **исправление багов**
    - [ ] улучшение парсера командной строки.
    - [ ] безопасный буфер клавиатуры.
    - [ ] скроллинг текста в консоли (сейчас текст уезжает вниз).

## v2.x — файловая система ext2 и gui
переход на более продвинутую файловую систему и графический интерфейс.

- [ ] **драйвер ext2**
    - [ ] чтение суперблока (superblock).
    - [ ] чтение таблицы дескрипторов групп (block group descriptor table).
    - [ ] парсинг инодов (inodes).
    - [ ] чтение файлов.
- [ ] **интеграция**
    - [ ] автоматическое определение фс (fat16 или ext2) при загрузке.
- [ ] **графический интерфейс**
    - [ ] оконный интерфейс.
    - [ ] драйвер мыши.
    - [ ] файловый менеджер с gui.

## v3.x — сети и драйверы
сетевая подсистема и поддержка устройств.

- [ ] **интернет**
    - [ ] драйвер сетевой карты (ne2000/virtio).
    - [ ] tcp/ip стек.
    - [ ] ping, http клиент, irc чат.
- [ ] **драйверы устройств**
    - [ ] usb (базовый).
    - [ ] звуковая карта.
    - [ ] поддержка разных процессоров.

## v4.x — системные возможности
ядро и управление памятью.

- [ ] **управление памятью**
    - [ ] реализация `malloc`/`free`.
    - [ ] виртуальная память.
    - [ ] защита памяти.
- [ ] **многозадачность**
    - [ ] запуск процессов.
    - [ ] планировщик.
    - [ ] межпроцессное взаимодействие.

## v5.x — 3d графика
трехмерная графика и игры.

- [ ] **программный 3d рендерер**
    - [ ] поддержка 3d моделей (obj формат).
    - [ ] текстуры и материалы.
    - [ ] opengl 1.1 совместимость (программная).
- [ ] **игры**
    - [ ] простые 3d игры.
    - [ ] поддержка геймпадов.

## v6.x — разработка и совместимость
инструменты и совместимость с другими ос.

- [ ] **средства разработки**
    - [ ] компилятор c для alphaos.
    - [ ] отладчик.
    - [ ] система сборки.
- [ ] **совместимость**
    - [ ] запуск linux программ.
    - [ ] эмуляция windows api.
    - [ ] виртуальные машины.
