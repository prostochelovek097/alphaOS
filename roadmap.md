# alphaos roadmap

план развития операционной системы.

## v1.x — стабилизация и fat16
текущий этап. улучшение работы с текущей файловой системой, делаем плеер видео.

- [ ] **поддержка папки (directories)**
    - [ ] реализация команды `cd` (смена директории).
    - [ ] чтение содержимого подпапок в `ls`.
    - [ ] поддержка путей вида `dir/file.txt`.
    - [ ] когда прописываем cd меняем / на /имя директории
- [ ] **исправление багов**
    - [ ] улучшение парсера командной строки.
    - [ ] безопасный буфер клавиатуры.
    - [ ] скроллинг текста в консоли (сейчас текст уезжает вниз).

## v2.x — файловая система ext2 и gui
переход на более продвинутую файловую систему и графический интерфейс.

- [ ] **драйвер ext2**
    - [ ] чтение суперблока (superblock).
    - [ ] чтение таблицы дескрипторов групп (block group descriptor table).
    - [ ] парсинг инодов (inodes).
    - [ ] чтение файлов.
- [ ] **интеграция**
    - [ ] автоматическое определение фс (fat16 или ext2) при загрузке.
- [ ] **графический интерфейс**
    - [ ] оконный интерфейс.
    - [ ] драйвер мыши.
    - [ ] файловый менеджер с gui.

## v3.x — мультимедиа (аудио и видео)
улучшенный аудиоплеер, интернет и драйвера устройств.

- [ ] **аудиоплеер**
    - [ ] поддержка mp3 (простой декодер).
    - [ ] поддержка wav (уже есть, улучшить).
    - [ ] микширование звуков.
    
- [ ] **драйверы устройств**
    - [ ] звуковая карта (ac'97 / sound blaster).
    - [ ] ускорение графики (базовое).

- [ ] **интернет**
    - [ ] драйвер сетевой карты (ne2000/virtio).
    - [ ] tcp/ip стек.
    - [ ] ping, http клиент, irc чат.

- [ ] **многозадачность**
    - [ ] запуск процессов.
    - [ ] планировщик.
    - [ ] межпроцессное взаимодействие.

## v4.x — архитектура ядра и сети
многозадачность и улучшение стеков сети.

- [ ] **менеджер виртуальной памяти**
    - [ ] Включить пейджинг (paging) вместо сегментации.
    - [ ] Выделять виртуальное адресное пространство процессам.
    - [ ] Защита памяти между процессами.

- [ ] **многозадачность**
    - [ ] структура PCB (Process Control Block) для хранения состояния.
    - [ ] простой планировщик (round robin).
    - [ ] системные вызовы для создания процессов (`fork`/`exec`-подобные).
    - [ ] переключение контекста (context switching).

- [ ] **сети и интернет (продвинутый стек)**
    - [ ] полноценный TCP-стек (установка соединения, потоки).
    - [ ] HTTP-клиент для загрузки простых веб-страниц.
    - [ ] простой текстовый браузер (например, для отображения .txt из сети).
    - [ ] поддержка DNS (разрешение доменных имён).

- [ ] **файловая система (улучшения)**
    - [ ] кэширование дисковых блоков в памяти.
    - [ ] фоновую запись данных на диск.
    - [ ] поддержка символических ссылокв ext2.

## v5.x — системные возможности
ядро, 3D графика, игры и видеоплеер.

- [ ] **3d графика**
    - [ ] программный 3d рендерер.
    - [ ] поддержка 3d моделей.
    - [ ] opengl 1.1 совместимость.

- [ ] **игры**
    - [ ] 2d/3d игровой движок.
    - [ ] поддержка геймпадов.
    - [ ] multiplayer по сети.

- [ ] **видеоплеер**
    - [ ] поддержка .3gp формата (как на sony ericsson).
    - [ ] поддержка mpeg-1 (простой стандарт).
    - [ ] базовый декодер h.263 (для .3gp).
    - [ ] вывод видео на экран (software decoding).
